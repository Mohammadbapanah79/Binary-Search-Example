# PHP Algorithm Exercises

## Project Description
This repository contains a collection of algorithm exercises implemented in PHP.  
The goal of this project is to practice programming skills, improve algorithmic thinking, and enhance problem-solving abilities.

## Algorithms and Exercises
- Binary Search  
- Linear Search  
- Sorting (Bubble, Selection, Insertion)  
- Recursion  
- Stack & Queue Examples  
- More exercises will be added over time

## Programming Language
- PHP (version 7 or higher)

## How to Run
To run any exercise, open the terminal in the file directory and use the following command:

```bash
php test.php
